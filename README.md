# Advent of Code 2020

You can find this year's problems [here](https://adventofcode.com/2020).


```bash
# Opens a new tab with the given day in browser
python .\aoc.py 2 open

# Run a specific day
python .\aoc.py 2 

# Run a specific day's tests
python .\aoc.py 2 test

# Run all tests
python -m unittest -v
```
